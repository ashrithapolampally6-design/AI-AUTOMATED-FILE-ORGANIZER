# Automated File Organizer

## 📌 Project Description
This project automatically organizes files into folders based on file type using Python.

## ⚙️ Features
- Sorts files into Images, Documents, Videos
- Automatically moves files into folders
- Easy to use automation tool

## 💻 Tech Used
- Python
- OS module
- Shutil module

## 🚀 How to run
python ORGANIZER.py
